# Pokedex_iOS
Allows a user to search a list of Pokemon (through String input or type selection) and see their statistics.

# Podfiles & Installation

```ruby
target 'Pokedex' do
  use_frameworks!
pod 'PokemonAPI'
end
```

```bash
pod install
```
Once podfiles have been installed, use the .xcworkspace filetype.
